# Biomedical Data Science Projects

This repository contains selected academic projects in biomedical data analysis, medical machine learning, and computational biology.

## Projects

- **Chest X-ray Classification with Transfer Learning**  
  Medical image classification using EfficientNetB0 and TensorFlow/Keras.

- **Cardiovascular Disease Statistical Analysis**  
  Exploratory statistics, PCA, factor analysis, and logistic regression for cardiovascular disease prediction.

- **Stochastic Gene Expression Modelling in COPASI**  
  Deterministic and stochastic simulation of the Peccoud & Ycart gene expression model, with Python-based trajectory analysis.
